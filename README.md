# Bus Fare Calculator

A simple web app to calculate bus fare based on travel distance, bus type, and student discount.

## Features
- Distance-based fare calculation
- Multiple bus types: Normal, AC, Express
- Student discount option
- Reset button
- Responsive UI

## Fare Rules
- First 5 km = Base Fare
- Additional km charged separately based on bus type

## Tech Used
- HTML
- CSS
- JavaScript
